# Compte rendu

Aujourd’hui, nous avons travaillé sur le bootstrap du projet T-DEV-810, en nous concentrant exclusivement sur la partie Arduino / IoT, conformément aux premières étapes du sujet.

Nous avons mis en place l’environnement matériel Arduino et testé plusieurs composants afin de valider :

- le câblage,
- la communication série,
- la stabilité globale du montage.

Nous avons ensuite travaillé sur la communication entre l’Arduino et le module ESP8266 via UART, en utilisant les commandes AT comme indiqué dans le bootstrap.
Cette étape nous a permis de vérifier :

- le bon fonctionnement de l’ESP8266,
- la transmission des commandes,
- la configuration du module en mode station pour une connexion Wi-Fi.
